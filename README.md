# symfony-task
1.Клонировать репозиторий  и перейти на ветку - symfony-task

2.Вызвать: composer install

3.Вызвать: php bin/console doctrine:database:create

4.Вызвать: php bin/console doctrine:migrations:migrate 

5.Вызвать: php bin/console server:run

6.Открыть страницу с приложение ( Обычно на  http://127.0.0.1:8000)
